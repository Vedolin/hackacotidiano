O objetivo deste projeto foi o desenvolvimento do módulo de interface com o usuário de um protótipo de bomba de infusão de insulina. Este projeto inseriu-se dentro de um projeto maior (financiado pela FAPESP) que tem como finalidade o desenvolvimento de um protótipo de bomba de infusão de insulina de baixo custo, para uso em nível nacional. O módulo de interface com o usuário é responsável pela interação do paciente com o sistema de controle da bomba de infusão de insulina. A interação ocorre por meio de quatro botões e um display LCD gráfico, além de avisos sonoros e vibração. O módulo de interface com o usuário foi desenvolvido em linguagem C e executado em microcontrolador da família MSP430. Durante esse projeto foi feito um estudo de casos sobre o funcionamento de uma bomba de infusão de insulina, assim como levantamento de dados de UX em relação as bombas já existentes no mercado. Após uma longa pesquisa com análise de usabilidade, com definição de requisitos do sistema usando SysML, foi definido uma interface. Quanto ao display, dividiu-se em duas linhas de desenvolvimento. Desenhei um modelo de display segmentado, no qual foi fabricado na China e visava o protótipo menor. E implementei uma biblioteca de interface gráfica para usar o GLCD que estava no protótipo maior (era na verdade um kit de desenvolvimento desenvolvido por nós mesmos com LEDs, GPIO, driver de motor , botões e etc.

Continuação do projeto iniciado em 2013, teve como objetivo o desenvolvimento de um protótipo de uma bomba de infusão de insulina de baixo custo. Durante o projeto foram feitos ajustes no protótipo V1, descrito anteriormente. Dentre os ajustes inclui refatoração do código fonte, excluindo gargalos de desempenho, projeto mecânico (CAD) de um novo sistema de movimentação do êmbolo da seringa com o motor de passo, revisão do projeto eletrônico. Ao final de um ano, foram feitos 1000 testes in-vitro, com 99,5% de confiabilidade, o que foi um sucesso, abrindo portas para o desenvolvimento do protótipo menor, que tem o tamanho de uma bomba de insulina usual. Importante ressaltar que os 99,5% ainda não é o ideal, precisam-se de ajustes ainda, pois se trata de um sistema crítico, e não pode haver falhas.
